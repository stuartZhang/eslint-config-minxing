# eslint-config-minxing
A group of ESLint presets for the minxing company.
# Setup
1. Global tools
    1. `npm install -g yo`
    1. `npm install -g generator-eslint`
1. Generate the eslint-plugin skeleton
    1. `yo eslint:plugin`
1. Generate the eslint-rules skeleton
    1. `yo eslint:rule`
